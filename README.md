## Caffeine

A little SPA built with React to order, track and update your daily dose of caffeine.

Goodies used in this project:
 - ES6
 - CSS Modules
 - Redux Ducks pattern
 - React router
 - Jest
 - ESlint
 - Webpack 4

Future enhancements:
  - React Intl
  - Redux saga with async actions
  - Reselect
